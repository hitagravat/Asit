# Asit

It is a college website.

**REQUIRE**: Node(22+)


## Quick Start

1. Clone The Repo
   ```
   git clone https://github.com/hitagravat/Asit
   ```
   
3. Install Nodejs 22+
   ```
   node -v
   ```

3. Move to Repository
   ```
   cd Asit
   ```

4. Install Dependency
   ```
   npm install
   ```

5. Run The Server
   ```
   npm run dev
   ```

6. Go to `localhost` at given port number to access the website.
   ```
   http://localhost:8080/
   ```


## Tech Stack (MERN)

  - Html
  - Css
  - Js
  - Express.js
  - Bootstrap.css
